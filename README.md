# Katalon Automation Framework

Automation testing framework menggunakan Katalon Studio untuk Web / API testing.

---

## Requirements

- Java JDK 8 atau 11
- Katalon Studio Latest Version
- Git
- Internet connection

---

## Installation / Setup Guide

### 1. Install Java

Check Java:

```
java -version
```

Jika belum install → download dari:
https://adoptium.net

---

### 2. Install Katalon Studio

Download:
https://katalon.com/download

Extract → jalankan Katalon Studio.

---

### 3. Clone Project

```
git clone https://github.com/USERNAME/katalon-automation-framework.git
```

---

### 4. Open Project di Katalon

- Open Katalon Studio
- File → Open Project
- Pilih folder project

---

### 5. Install Dependencies

Katalon otomatis download dependency saat project dibuka.

Jika error:

```
Project → Clean Up → Build Project
```

---

### 6. Run Test

#### Run UI Test

- Test Cases → pilih test case
- Klik Run

#### Run Test Suite

- Test Suites → pilih suite
- Klik Run
